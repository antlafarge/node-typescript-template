APP
===

# Init project
```bash
npm init -y
npm install express
npm install --save-dev @eslint/js @types/express @types/node eslint globals nodemon ts-node typescript typescript-eslint
```

# Setup

## First time
```bash
npm install
```

## Build
```bash
npm run build
```

## Start
```bash
npm run production
npm run test
npm run staging
npm run development
```
