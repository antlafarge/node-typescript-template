APP
===

# Init project
`npm init -y`
`npm install express`
`npm install --save-dev @eslint/js @types/express @types/node eslint globals nodemon ts-node typescript typescript-eslint`

# Setup

## First time
`npm install`

## Build
`npm run build`

## Start
`npm run production`  
`npm run test`  
`npm run staging`  
`npm run development`
