# APP

# Init project

```bash
npm init -y
npm install -D @eslint/js @types/node eslint nodemon prettier tsx typescript typescript-eslint
```

# Setup

## First time

```bash
npm install
```

## Build

```bash
npm run build
```

## Tools
```bash
npm run lint
npm run lint:fix
npm run format
```

## Start

```bash
npm run development
npm run test
npm run staging
npm run production
```
